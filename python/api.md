## THE API


Everything is json

### Gun

- type: gun_turn_right
- msg: float between 0 and 1

- type: gun_turn_left
- msg: float between 0 and 1


- type: gun_shoot
- msg: nothing

### Thrust

- type: turn_right
- msg: float between 0 and 1

- type: turn_left
- msg: float between 0 and 1

- type: thrust_forward
- msg: float between 0 and 1
